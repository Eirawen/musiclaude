# 010: New Premise, New Instrumentation

**Date:** 2026-03-17
**Status:** Complete
**Result:** Best results in project history. v3 canonical profile wins. Cello >> clarinet.

## Question

Is the clarinet writing weakness instrument-specific? Does v3 canonical profile work
under minimal prompting with different instrumentation?

## Answer

**YES and YES.** Cello+piano produced scores of 85-90 (avg 87.3) vs clarinet+piano's
50-83 (avg 71). The v3 canonical profile, which produced "clownhouse" music in experiment
007, won with 90/100 — the highest score in the project.

## Key Findings

1. **Instrumentation is the biggest lever** — cello+piano avg 87.3 vs clarinet+piano
   avg 71. Worth more than any profile or workflow change.
2. **v3 canonical profile rehabilitated** — avg 89 vs v1's 85.5. It never had bad
   targets; it was trapped in a bad pipeline writing for a bad instrument.
3. **Workflow doesn't matter much** — exp5 (87.5) ≈ exp9 (87.0) when instrumentation
   is right.
4. **90/100 is the new ceiling** — listener said "really good," praised call-and-response,
   continuous flow, specific standout moments.
